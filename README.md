# Portfolio Card Generator
A simple web application that allows users to create and preview a personal portfolio card in real time.

## Live Demo
[https://still-purrfect.github.io/portfolio-card-generator/](https://still-purrfect.github.io/portfolio-card-generator/)

## Features
* Real-time card preview
* Multiple theme options
* Custom color selection
* Local storage support (auto-save)
* Reset form functionality
* Download card as image
* Responsive design for all devices

## Built With
* HTML5
* CSS3
* JavaScript (Vanilla)
* html2canvas (for image export)

## How It Works
* Enter your details in the form
* The preview card updates instantly
* Data is saved automatically in the browser
* Download your card as an image when ready
